# Скрипт для поиска арбитражных возможностей на криптовалютных биржах

Этот скрипт получает цены на Bitcoin (BTC) с трёх различных бирж (Binance, Coinbase, Kraken) и находит арбитражные возможности.

## Что делает скрипт?

1. **Получает цены с бирж**: Скрипт автоматически запрашивает текущие цены BTC/USD с трёх крупных бирж:
   - Binance (BTC/USDT)
   - Coinbase (BTC/USD)
   - Kraken (BTC/USD)

2. **Находит арбитражные возможности**: Определяет, где можно купить дешевле и продать дороже:
   - Находит самую низкую цену продажи (ask) - где купить
   - Находит самую высокую цену покупки (bid) - где продать
   - Рассчитывает потенциальную прибыль в процентах

3. **Обрабатывает ошибки**: Если одна из бирж недоступна, скрипт продолжит работу с остальными

## Установка

1. Убедитесь, что у вас установлен Python 3.6 или выше:
```bash
python --version
```

2. Установите необходимые зависимости:
```bash
pip install -r requirements.txt
```

Или установите напрямую:
```bash
pip install requests
```

## Использование

Запустите скрипт из командной строки:

```bash
python crypto_arbitrage.py
```

```

Kraken: Bid=50002.00, Ask=50012.00

Арбитражная возможность:
Купить на Binance по цене 50010.00 и продать на Coinbase по цене 50005.00
Потенциальная прибыль: -0.01%
✗ Нет выгодной арбитражной возможности (цена продажи <= цена покупки)
```

## Объяснение результатов

- **Bid (цена покупки)** - цена, по которой биржа готова купить у вас BTC
- **Ask (цена продажи)** - цена, по которой биржа продаёт вам BTC
- **Арбитражная возможность** - когда можно купить на одной бирже дешевле и продать на другой дороже
- **Потенциальная прибыль** - расчётная прибыль в процентах (без учёта комиссий)

## Важные замечания

⚠️ **Внимание:**
- Скрипт использует публичные API, не требующие авторизации
- Для работы необходимо интернет-соединение
- Арбитражные возможности на криптовалютных рынках обычно очень малы и быстро исчезают
- В расчёте прибыли НЕ учитываются:
  - Комиссии бирж
  - Лимиты на вывод средств
  - Время на перевод средств между биржами
  - Проскальзывание цены (slippage)

## Устранение проблем

Если скрипт выдаёт ошибки:

1. **Ошибка импорта `requests`**: Установите библиотеку
   ```bash
   pip install requests
   ```

2. **Ошибки подключения к API**: Проверьте интернет-соединение и доступность бирж

3. **Timeout ошибки**: Увеличьте таймаут в коде (по умолчанию 10 секунд)

## Кастомизация

Вы можете изменить:
- Торговую пару (измените URL в начале файла)
- Таймауты запросов (параметр `timeout=10` в функциях)
- Добавить другие биржи (создайте новую функцию по аналогии)

## Лицензия
